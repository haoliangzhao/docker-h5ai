# docker-h5ai Fork Changes

This README lists the changes in this fork compared with
[awesometic/docker-h5ai](https://github.com/awesometic/docker-h5ai).

## Changes

- Adds a light and dark theme switch with system theme detection and saved preferences.
- Uses a neutral dark gray color palette for dark mode.
- Makes the sidebar push the page content instead of covering it.
- Makes the file selection icon smaller.
- Adds a `Copy link` button below the QR code in the Info panel.
- Hides the Tree and Info sidebar options on narrow screens.
- Adds a Docker Compose development mode with live-mounted source files.
